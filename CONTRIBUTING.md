# Contributing

The current scope accepts changes to repository inventory, JavaScript/TypeScript parser adapters, normalized facts, the Secure Engine-owned evidence graph, deterministic rules, suppressions, the local parse cache, the public `secure-json-v1` contract, SARIF export, deterministic baselines, private local history, native UI boundaries, and Fedora RPM packaging. Other-language parsers, AI integration, automatic fixes, cloud services, telemetry, AppImage, and hosted services remain out of scope.

Run the checks in `docs/development.md` before submitting a change. Contributions are made under the MIT License and must include a Developer Certificate of Origin sign-off (`git commit -s`), certifying that the contributor has the right to submit the work under this license.

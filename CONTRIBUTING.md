# Contributing

The current scope accepts changes to repository inventory, JavaScript/TypeScript parser adapters and normalized facts, the local parse cache, the public `secure-json-v1` contract, and its interfaces. Semantic graphs, other-language parsers, AI integration, automatic fixes, and vulnerability rules are intentionally deferred.

Run the checks in `docs/development.md` before submitting a change. Contributions are made under the MIT License and must include a Developer Certificate of Origin sign-off (`git commit -s`), certifying that the contributor has the right to submit the work under this license.

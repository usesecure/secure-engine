# Phase 6.11 tranche 3 independent fixtures

The executable fixtures in
`crates/secure-engine/tests/generalization_phase611_tranche3.rs` derive from two
general invariants:

1. a constructed redirect is safe only when a dominant terminating check proves
   the complete fixed origin of the same unreassigned URL value; and
2. an outbound finding follows only the exact static property selected from an
   unambiguous object and passed to the existing sensitive argument.

The matrix covers JavaScript, JSX, TypeScript, and TSX; direct, property-bound,
helper, unique inter-file, alias, nested, adversarial, mutation, and metamorphic
forms. It pairs recognized vulnerable flows with clean or near-miss controls
that change the structural invariant. Identifiers, domains, literals, paths,
comments, and structures are Engine-owned and independent of known development
or evaluation material.
